# Para Code 更新履歴

Para Code が本家 VS Code に加えた変更の一覧です。新しいバージョンが上に来ます。

## 未リリース

### 修正

- Word文書ビューアで、複数ページの文書を開いた際にページごとに用紙の幅が食い違って見える（一部のページだけ異常に広い/狭い）問題を修正しました
- Word文書ビューアで、斜めの罫線・区切り線が表示されないことがある問題を修正しました
- Word文書ビューアで、縦書きの見出しセルが正しく縦書き表示されず、横に大きくはみ出すことがある問題を修正しました

## paracode-17（2026-07-07）

### 修正

- Word文書ビューアで、番号付きリスト・箇条書きの番号/記号が一切表示されない問題を修正しました（表示用スタイルがブラウザのセキュリティ設定で無効化されていたのと、番号表示ライブラリ自体の不具合が原因でした）
- Word文書ビューアで、文字の色・フォントなど細かい書式が正しく反映されないことがある問題を修正しました
- Word文書ビューアで、幅の広い表がページの外にはみ出して背景色で隠れてしまう問題を修正しました

## paracode-16（2026-07-06）

### 新機能

- PDFファイルのレンダリング表示に対応しました。PC版はPDFを開くと専用ビューアで表示され、ズーム（Ctrl/Cmd+ホイール対応）・ページ番号表示・幅合わせができます（日本語PDFの非埋め込みフォントにも対応）
- Word文書（.docx）のレンダリング表示に対応しました。PC版は.docxを開くと専用ビューアでページ風のレイアウトで表示されます（見出し・表・画像・ヘッダー/フッターに対応）
- Para Code Mobile のファイルタブでもPDFを開けるようになりました（ピンチズーム・テキスト選択対応。20MBまで）
- Para Code Mobile のエージェントチャットで、Claude Code の選択式の質問（AskUserQuestion）が質問カードとして表示され、選択肢をタップしてそのまま回答できるようになりました（回答済みの質問はグレー表示）
- Para Code Mobile: アプリを開いていなくても（バックグラウンド・終了時でも）、エージェントの質問や作業完了がプッシュ通知で届くようになりました。通知本文はPCとスマホの間で暗号化されたまま配送され、中継サーバーからは内容が見えません（要: アプリの再ビルドとリレーの更新）
- ワークスペース一覧のワークツリー（スペース）を右クリックから削除できるようになりました（確認ダイアログ付き。未コミット変更がある場合は強制削除するかを追加確認します）
- ステータスバーのモバイル接続表示をクリックするとメニューが開き、デバイスのペアリング・接続解除・リレーの無効化（接続を切る）ができるようになりました

### 修正

- ステータスバーの Bongo Cat が表示されないユーザーがいた問題を修正しました。Bongo Cat の表示に必要なプロダクトアイコンテーマを既定で有効化するようにしました（settings.json でプロダクトアイコンテーマを自分で選んでいる場合はそちらが優先されます）
- 通知設定の通知音サンプル再生ボタンを押しても音が鳴らない問題を修正しました（配布版に音声ファイルが同梱されていませんでした）
- 通知設定ダイアログでボタンや入力欄を操作するとスクロール位置が飛ぶことがある問題を修正しました
- 内蔵ブラウザでページを開いたままワークスペースを切り替えると、ページの一部が画面に残って表示されることがある問題を修正しました

### 修正（Para Code Mobile）

- PC側で選択中でないワークスペースのターミナル（エディタエリアで開いているClaude Code等）が、ターミナルタブ・エージェントタブに表示されない問題を修正しました
- アプリから新規ターミナルを作成したとき、PC側のターミナルパネルに表示されない問題を修正しました
- ターミナルタブで Claude Code / Codex などのTUI表示が開いた直後に乱れる問題を修正しました（画面状態を色・カーソル位置ごと復元するようにしました）
- PC側のモバイル接続状態表示（フッター右下）が、起動直後や再接続中に消えてしまうことがある問題を修正しました

### 修正（Para Code Mobile）

## paracode-14（2026-07-06）

### 改善（Para Code Mobile）

- ソース管理タブのコミット履歴で、コミット行をタップするとそのコミットの変更ファイル一覧を展開表示できるようになりました（ブラウザで開くのは行末のアイコンから）
- ロック解除後の猶予時間内（約10分）は、アプリスイッチャーに切り替えてもロック画面を表示しないようにしました

## paracode-13（2026-07-06）

### 新機能（Para Code Mobile）

- アプリ起動時に Face ID / Touch ID（または端末パスコード）でロックを解除するようになりました。他のアプリへ切り替えても、離れてから10分以内の復帰なら再認証は不要です
- エージェントタブの会話が見やすくなりました。ツール実行と思考の過程は「思考 ・ ツールN件」のような1行に自動で折りたたまれ、タップで詳細を展開できます（CLI版と同様の表示方針）

### 改善（Para Code Mobile）

- ブラウザタブの画面更新を大幅に高速化しました。PC側の再描画に合わせて即座にフレームが届くようになり（従来は約0.25秒間隔の定期取得）、スクロールや画面遷移の追従が滑らかになります。ウィンドウ最小化中などPC側で描画が起きない間は、自動的に従来の定期取得へ切り替わります
- Excelの表示を高速化しました。行数の多いシートは先頭から順次描画して最初の画面をすぐ表示し、一度表示したシートはPC側にキャッシュされるため、シートタブの往復や開き直しが即座になります。隣のシートも先読みします

### 修正（Para Code Mobile）

- ブラウザタブで、PC側のウィンドウをリサイズした後などにタップ位置がズレることがある問題を修正しました
- ソース管理タブでコミット履歴が表示されなかった問題を修正しました。直近10件を常に表示し、「さらに読み込む」で過去のコミットを10件ずつ追加表示できます。履歴の取得に失敗した場合はエラーを表示します
- 大きなExcelシートを開くと「読み込み中…」のまま進まなくなる問題を修正しました。表示データを圧縮してサイズ上限内に収まるようにし、失敗した場合もエラー内容を表示します
- Excelのシートがまれに異様に横長になったり、罫線や斜線（図形）が表示されないことがある問題を修正しました（表示用スタイルの読み込み失敗時に壊れた表示のまま出さないようにしました）

### 改善（Para Code Mobile）

- Excelの表示をシート単位の読み込みに刷新しました。シートタブ（ネイティブのタブ）で切り替えるたびに該当シートだけを取得するため、シート数の多い大きなブックでも省略されずに全シートを表示できます
- Excelのセル幅・レイアウトがPC版と食い違うことがある問題を修正しました（テーマ色の引き継ぎと、スマホWebViewの文字自動拡大の無効化。ファイルタブ・ソース管理の差分の両方に適用）
- テキスト検索の結果をタップしたとき、該当行まで自動スクロールしてハイライト表示するようになりました
- PCでclaude / codexを起動した時点でエージェントタブが自動的にセッションを認識するようになりました（「セッションが見つかりません」のまま止まる問題を修正）

## paracode-11（2026-07-06）

### 改善（Para Code Mobile）

- ファイルタブの検索を全面刷新しました。VS Code本体と同じ検索エンジン（ripgrep）を使い、ワークスペース全階層からのファイル名検索と、ファイル内容のテキスト全文検索（「名前 / 内容」切り替え）ができるようになりました。.gitignore を尊重し、深い階層のファイルも確実に見つかります
- 大きなExcelファイルがアプリで表示できなかった問題を修正しました。転送を分割方式に刷新してサイズ上限を大幅に緩和（約900KB→6MB）し、生成待ちのタイムアウトも延長しました
- PCでエージェント（claude / codex）を起動済みなのに、エージェントタブで「セッションが見つかりません」と表示されることがある問題を修正しました（アプリ起動前から動いているセッションも自動で検出します）
- エージェントタブの応答をMarkdownで表示するようになりました（太字・コードブロック・インラインコード・見出し・箇条書き）
- エージェントが作業中のとき、チャット末尾に「考え中…」のアニメーションを表示するようになりました
- ファイルビューアで開ける上限を256KB→1MBに拡大しました

## paracode-10（2026-07-06）

### 新機能（Para Code Mobile）

- 「エージェント」タブを追加しました。PCのターミナルで動いている claude / codex の会話を、チャット形式（吹き出し・ツール実行の折りたたみ表示）でスマホから確認できます。PC側はターミナル（TUI）のまま変わりません
- エージェントタブからメッセージを送信でき、PCのターミナルに入力したのと同じように届きます。エージェントが確認（許可/拒否）を求めているときは、承認バーのボタンでそのまま応答できます
- 接続が一時的に切れても、再接続時に会話の続きから自動で同期されます

### 改善（Para Code Mobile）

- Excel（.xlsx / .xlsm）に対応しました。ファイルタブではPC版ビューアと同じ見た目（セルの色・罫線・結合・図形・シートタブ）で表示され、ソース管理の差分でも HEAD と作業ツリーをセル単位の色分けで比較できます。どちらもピンチで拡大縮小できます
- ソース管理の差分ビューアで、.md / .html のときに「Diff / レンダー」を切り替えられるようになりました（Excelも同様にレンダー表示へ切り替え可能）
- アプリ再起動や復帰後に接続されないことがある問題を修正しました（接続処理にタイムアウトを設け、フォアグラウンド復帰時に自動で繋ぎ直します）
- 未接続のあいだは、ホーム以外のタブに接続状態と再接続ボタンを表示するようにしました（未接続のまま操作できてしまう状態を防止）
- ターミナルでキーボードを開いたときに、出力の最下部（プロンプト行）が隠れて見えなくなる問題を修正しました
- ホームの「接続中のPC」カード右上に切断/接続ボタンを追加しました

## paracode-8（2026-07-05）

### 新機能

- **モバイル遠隔操作（Para Code Mobile・プレビュー）**: iPhone アプリと一度ペアリングしておくと、別のネットワークからでも PC の Para Code を遠隔操作できるようになりました（設定 `paradis.mobile.enabled`、既定はオフ）
  - 「Para Code: モバイルデバイスを接続」コマンドで QR コードをパネル表示し、アプリでスキャン（またはリンク貼り付け）→ 6桁コードを突き合わせて承認するだけで接続できます（通信は端末間でエンドツーエンド暗号化され、中継サーバーは内容を読めません）
  - アプリはホーム／ターミナル／ソース管理／ファイル／ブラウザの5画面構成で、上部のワークスペースバーで対象リポジトリを切り替えられます
  - ターミナル: PC の実ターミナルをタブごとにミラー表示・入力でき、開いた時点で直前の出力（スクロールバック）も見えます
  - ソース管理: 変更ファイル一覧・差分プレビュー・コミット・最近のコミット履歴をスマホから操作できます
  - ファイル: ワークスペース内のフォルダを辿ってテキストファイルをプレビューできます（読み取り専用）
  - ブラウザ: PC 側 para-browser のページ一覧から選んでスマホに同期表示し、タップ・スクロール・戻る/進む/再読み込みを送れます
  - エージェント（Claude Code / Codex）が質問・許可待ちになったときや作業を完了したときに、スマホへ通知が届きます（接続中の場合）。ホームの「回答する」から該当ターミナルへ直行できます
  - 「Para Code: モバイルデバイスの管理」コマンドでペアリング済みデバイスの一覧・失効ができます
  - PC 側の端末鍵は OS のセキュアストレージ（safeStorage）で暗号化して保存します
- **コマンドプリセット**: よく使うコマンドの組み合わせ（例: `bun run dev` 一式）を登録して、ターミナルタブバー右側のボタン・コマンドパレットからワンクリックで実行できるようになりました
  - 起動モードを選べます（アクティブなターミナルで連結実行 / 新しいターミナル / コマンドごとに新しいターミナル / 分割して並べる）
  - ボタンのアイコン（一覧から選択）・作業ディレクトリ・対象リポジトリの条件を指定できます
  - 保存先は「ユーザー設定（全リポジトリ共通、設定 `paradis.terminal.presets`）」と「リポジトリ（`.paracode.json` — コミットすればチームや worktree 全体で共有）」の2レベル
  - 管理ダイアログは、ターミナルタブバー右側の工具アイコン・「Para Code: コマンドプリセットを管理」コマンド・設定画面の該当項目のリンクのどこからでも開けます
- **ccusage 使用量ダッシュボード**: ステータスバー右端（通知ベルの左）に今日のエージェント利用コストが表示され、クリックするとエディタタブでダッシュボードが開くようになりました（コマンド「Show ccusage Dashboard」でも開けます）
  - 期間（今日/昨日/今週/先週/7日/30日/90日/カスタム範囲）とエージェント（Claude Code / Codex / Gemini）で絞り込みでき、日別コストのモデル別内訳・トークン推移・モデル別/プロジェクト別コスト・直近セッション・現在の5時間ブロック（消費速度と終了時予測）をグラフで確認できます。表示単位はDaily/Weeklyを手動で切り替えられます（選択期間が7日未満のときはWeeklyを選べません）
  - 集計には [ccusage](https://ccusage.com) CLI を使います。未インストールでも `npx` 経由で自動実行されます（設定 `paradis.ccusage.executablePath` で実行ファイルを固定可能、ステータスバー表示は `paradis.ccusage.statusBar.enabled` でオフにできます）
- **自然言語からスペース（worktree）を作成**: Workspaces ビューの ✦ ボタンから、「何をしたいか」を書くだけで新しい worktree の作成 → 切り替え → エージェント CLI（Claude Code / Codex / Gemini、設定 `paradis.workspaceSwitch.agents` で追加可）の起動までを一括で行えるようになりました
  - ブランチ名は手入力 > プロンプトからの自動生成（Copilot サインイン時）> 日時ベース、の順で決まります
  - プロンプトを空にすれば通常の worktree 作成としても使えます
  - プリセットの「自動実行」を有効にすると、スペース作成直後に dev サーバー等を自動起動できます（リポジトリ定義のものは初回に内容を確認します）

### 修正

- アプリを再起動するたびに左サイドバーが閉じた状態で開いてしまう問題を修正しました（終了時の開閉状態が正しく復元されます。開発環境では再現せずパッケージ版でのみ起きる問題でした）
- 「ブラウザページをエージェントと共有」ダイアログと通知設定ダイアログの周囲に、常に青い枠線が表示されていた問題を修正しました

### 改善

- 「ブラウザページをエージェントと共有」ダイアログの MCP 接続設定を「MCP接続設定」タブに集約しました（ターミナルペインタブ側の重複表示をなくし、ペイン一覧を広く使えるようにしました）
- Para Code 独自のダイアログ（ブラウザ共有・通知設定・コマンドプリセット管理・スペース作成・ブックマークなど）の配色・角丸・影を、本体の設定ダイアログと同じデザインに統一しました（ウィンドウ透過時は他の領域と同じように背景が透けます）
- 通知設定（歯車メニュー → 通知設定）に、エージェントのデスクトップ通知（OS の通知センター）の設定を追加しました
  - デスクトップ通知のオン/オフと、通知するイベント（対応待ち / 作業完了）を選べます
  - 「Para Code を見ている間も通知する」を有効にすると、いま開いているスペースのイベントもフォーカス中に通知されます（既定ではフォーカス中は通知されません）

## paracode-1（2026-07-03）

Para Code の初回リリースです。VS Code 1.128 をベースに、以下の独自機能を追加しています。

### 新機能

- **ワークスペースの即時切り替え**: 左サイドバーの Workspaces ビューに複数のリポジトリや git worktree を登録し、ワンクリック（または ⌃⌘1〜9）で切り替えられます。開いていたエディタ・ターミナル・コミットメッセージの下書きは切り替え先ごとに保持されます。スペースごとの色分け・名前変更・エージェント稼働状態の表示にも対応
- **ターミナルの 2D グリッド分割**: ターミナルを上下左右の 4 方向に分割し、ドラッグ & ドロップで自由に配置できます
- **内蔵ブラウザ**: エディタ内でブラウザを開けます（Toggle Browser）。ブックマークバー、AI エージェントとの連携（ページやターミナルをエージェントに割り当て）に対応
- **レンダリングファイルビューア**: HTML / Markdown / Excel（xlsx）をレンダリングした状態で表示できます。Excel はセル単位の git 差分表示に対応
- **通知**: エージェントの作業完了などを通知音と Aivis の音声読み上げで知らせます
- **リソースモニター**: タイトルバーに CPU / メモリ使用率を表示します
- **ターミナル履歴補完**: シェルを横断した永続コマンド履歴から補完できます（↓キーで候補を表示）
- **ウィンドウ透明度**: 背景だけを半透明にしてデスクトップを透かせます（設定の「Para Code」セクション）
- **セカンダリアクティビティバー**: 右サイドバーにもアクティビティバーを追加し、既定で Source Control を配置

### 変更

- アプリアイコンとアクセントカラーを Para Code 独自のデザインにしました
- 初回起動時に既定の拡張機能セットを自動インストールします
- 独自の配布・自動アップデート基盤に接続します（macOS / Windows / Linux）
