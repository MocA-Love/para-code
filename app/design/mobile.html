画面と機能の対応

ペアリングPC側Para Codeでコマンドを実行し、QR/6桁コードで一度だけ紐付け。以後は別ネットワークからでもリレー経由(E2E暗号化)で接続。
ホーム接続中PCの状態と、開いている全ワークスペースをグループ表示。各ワークスペース内のClaude Code / Codexの動作状況を一覧し、質問待ちのエージェントには即座に回答できる。「開く」でそのワークスペースの作業画面へ。
ワークスペース切り替えターミナル・ソース管理・ファイル・ブラウザの4画面は上部のワークスペースバーで対象を切り替える。切り替えは全画面で連動し、PC側のマルチワークスペースと1対1で対応。応答待ちがあるワークスペースにはバッジを表示。
ターミナル選択中ワークスペースのターミナルタブをミラー表示・入力。エージェント応答待ちのタブは赤ドットで示す。修飾キー行でEsc/Ctrl等も送信可能。
ソース管理変更ファイル一覧・差分プレビュー・コミット。リポジトリ/ブランチ切替に対応。
ファイルワークスペースのファイルツリーを閲覧、Markdown等はプレビュー表示。
ブラウザPC側のブラウザビュー(para-browser)と同期し、エージェントが見ているページをスマホでも確認・操作。
通知Claude Code / Codexの質問・タスク完了・エラーをプッシュ通知で受信。バナーからワンタップでターミナルへ。
